# i-SEAL GitHub Pages Website Package

This is a simple static website package for **GitHub Pages**.

## Files
- `index.html` — main page content
- `styles.css` — design and layout
- `script.js` — small footer year script

## Fastest deployment on GitHub Pages

1. Create a new **public repository** on your GitHub account.
2. Upload these three files to the repository root.
3. Open the repository settings.
4. Go to **Pages**.
5. Under **Build and deployment**, choose:
   - **Source**: Deploy from a branch
   - **Branch**: `main`
   - **Folder**: `/ (root)`
6. Save.
7. Wait a little, then your site will be published at a GitHub Pages URL.

## Where to edit the content
Open `index.html` in any text editor and change:
- lab description
- professor name and profile text
- member names
- publications
- news
- contact information
